2. Dados que Coletamos via LinkedIn API

2.1 Dados Coletados Através da Autenticação LinkedIn

Quando você faz login em nosso aplicativo usando sua conta LinkedIn, podemos coletar os seguintes dados, conforme as permissões (scopes) que você autoriza:

  • Dados de perfil básico: Nome, foto de perfil, URL do perfil LinkedIn, headline
  • Dados de contato: Endereço de email (se autorizado)
  • Dados profissionais: Cargo atual, empresa, experiência profissional (se autorizado)
  • ID do LinkedIn: Identificador único do usuário na plataforma LinkedIn

Importante: Nós só coletamos os dados que você explicitamente autoriza através do processo OAuth 2.0 do LinkedIn. Você pode revogar essa autorização a qualquer momento nas configurações da sua conta LinkedIn.

2.2 Dados de Uso do Aplicativo

Quando você utiliza nosso aplicativo, podemos coletar:

  • Dados de autenticação: Tokens de acesso, timestamps de login
  • Dados de interação: Funcionalidades utilizadas, tempo de uso
  • Dados técnicos: Endereço IP, tipo de dispositivo, navegador

2.3 Dados que NÃO Coletamos

Não coletamos:

  • Mensagens privadas do LinkedIn
  • Lista completa de conexões (salvo quando autorizado explicitamente)
  • Dados de recrutamento ou candidaturas
  • Informações de pagamento do LinkedIn Premium

4. Compartilhamento de Dados

Não vendemos seus dados pessoais coletados via LinkedIn API. Compartilhamos dados apenas nas circunstâncias descritas abaixo:

4.1 LinkedIn Corporation

Os dados coletados através da API do LinkedIn são processados de acordo com os Termos de Uso e Política de Privacidade do LinkedIn. O LinkedIn atua como controlador conjunto dos dados que você autoriza através da API.

4.2 Serviços de Hospedagem e Infraestrutura

  • Plataformas de hospedagem: Serviços de cloud computing para armazenar dados do aplicativo
  • Serviços técnicos: CDN, monitoramento, logs e segurança (quando aplicável)

4.3 Requisitos Legais

Podemos compartilhar dados quando:

  • Exigido por lei, ordem judicial ou autoridade regulatória
  • Necessário para proteger nossos direitos, propriedade ou segurança
  • Prevenir fraudes ou atividades ilegais
  • Solicitado pelo LinkedIn para verificação de compliance

4.4 Transferências Internacionais

Seus dados podem ser transferidos e processados em países fora do Brasil/UE, incluindo servidores do LinkedIn Corporation. Quando aplicável, utilizamos cláusulas contratuais padrão da UE (SCCs) e outras salvaguardas conforme GDPR Art. 46 e LGPD Art. 33.

6. Seus Direitos (LGPD, GDPR, CCPA)

Conforme a legislação aplicável, você tem os seguintes direitos sobre seus dados coletados através da API do LinkedIn:

  • Acesso: Solicitar confirmação e acesso aos seus dados coletados via API
  • Correção: Solicitar correção de dados incompletos ou inexatos
  • Eliminação: Solicitar exclusão de dados (quando aplicável por lei)
  • Portabilidade: Solicitar transferência de dados a outro fornecedor
  • Oposição: Opor-se ao tratamento em determinadas circunstâncias
  • Revogação de autorização: Revogar acesso do app à sua conta LinkedIn a qualquer momento
  • Limitação: Solicitar limitação do tratamento
  • Informação: Ser informado sobre compartilhamentos com terceiros
  • Não discriminação (CCPA): Não ser discriminado por exercer direitos de privacidade

6.1 Como Exercer Seus Direitos

Para exercer qualquer um desses direitos, entre em contato através do email especificado na seção de contato abaixo. Responderemos dentro do prazo legal (LGPD: 15 dias; GDPR: 1 mês).

Revogação de Acesso LinkedIn: Você também pode revogar o acesso do nosso aplicativo à sua conta LinkedIn diretamente nas configurações de privacidade do LinkedIn, em "Apps e Serviços" > "Permissões de Apps".

7. Proteção de Dados de Menores

COPPA 2024/2025 Compliance: Nossos serviços não são direcionados especificamente a menores de 13 anos. Não coletamos intencionalmente dados pessoais de crianças. Se descobrirmos que coletamos dados de menor sem consentimento parental, tomaremos medidas para eliminar essas informações.

Se você é pai/mãe ou responsável e acredita que coletamos dados de seu filho sem consentimento, entre em contato imediatamente.

8. Transparência de IA (Lei Felca nº 15.211/2025)

Conformidade com Lei Felca: Informamos que utilizamos inteligência artificial para auxiliar na criação de conteúdo, incluindo:

  • Geração de rascunhos e sugestões de conteúdo
  • Revisão gramatical e otimização SEO
  • Análise de dados e insights de audiência
  • Tradução e adaptação de conteúdo

Todo conteúdo é revisado, validado e aprovado por humanos antes da publicação. A IA é utilizada como ferramenta de suporte, não como substituto de julgamento humano crítico.

9. Retenção de Dados

Retemos seus dados pessoais coletados através da API do LinkedIn apenas pelo tempo necessário para as finalidades descritas nesta política, salvo exigência legal ou regulatória. Os critérios de retenção incluem:

  • Tokens de acesso OAuth: Conforme configuração do token (geralmente 60 dias para refresh tokens)
  • Dados de perfil: Enquanto você mantiver o acesso do app à sua conta LinkedIn
  • Dados de analytics: Até 26 meses (conforme configuração padrão)
  • Dados legais: Conforme período prescricional aplicável (geralmente 5 anos)

Exclusão: Quando você revoga o acesso do app à sua conta LinkedIn, solicitamos a exclusão dos dados armazenados em nossos sistemas dentro de 30 dias, salvo obrigação legal de retenção.

10. Segurança de Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados coletados via API do LinkedIn:

  • Criptografia: HTTPS/TLS para todas as conexões e criptografia de tokens OAuth
  • Controle de acesso: Acesso restrito a dados pessoais e tokens de autenticação
  • OAuth 2.0: Uso de protocolo OAuth 2.0 seguro para autenticação via LinkedIn
  • Monitoramento: Monitoramento contínuo de atividades suspeitas e tentativas de acesso não autorizado
  • Backups: Backups regulares com proteção adequada e criptografia
  • Atualizações: Manutenção regular de sistemas e segurança conforme melhores práticas

Aviso: Nenhum sistema é 100% seguro. Não podemos garantir absoluta segurança, mas adotamos práticas industry-standard para proteger seus dados e tokens de autenticação.

11. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos usuários sobre alterações significativas através de:

  • Aviso no aplicativo
  • Comunicação por email (quando aplicável)
  • Notificação através do LinkedIn (quando permitido)

Recomendamos revisar esta política regularmente. A data de "Última atualização" no topo indica quando esta política foi revisada pela última vez.

Perguntas Frequentes

Quais dados o app coleta através da API do LinkedIn?

O app coleta apenas os dados que você autoriza através do processo OAuth 2.0 do LinkedIn, como nome, foto de perfil, email (se autorizado), cargo atual e empresa. Não coletamos mensagens privadas, lista completa de conexões ou dados de recrutamento. Consulte a seção 2 para detalhes.

Como posso revogar o acesso do app à minha conta LinkedIn?

Você pode revogar o acesso a qualquer momento nas configurações de privacidade do LinkedIn, em "Apps e Serviços" > "Permissões de Apps". Após revogar, seus dados serão excluídos de nossos sistemas em até 30 dias, salvo obrigação legal.

O app vende meus dados do LinkedIn?

Não, não vendemos seus dados pessoais coletados via LinkedIn API. Compartilhamos dados apenas com serviços essenciais de hospedagem e quando exigido por lei. O LinkedIn atua como controlador conjunto dos dados que você autoriza através da API.

Como posso exercer meus direitos LGPD ou GDPR sobre os dados do app?

Você pode exercer seus direitos de acesso, correção, eliminação, portabilidade e oposição enviando um email para privacidade@canalqb.com.br. Responderemos dentro do prazo legal (15 dias para LGPD, 1 mês para GDPR). Consulte a seção 6 para mais informações.

Por quanto tempo o app retém meus dados do LinkedIn?

Retemos dados enquanto você mantiver o acesso do app à sua conta LinkedIn. Tokens OAuth são retidos conforme sua configuração (geralmente 60 dias). Ao revogar o acesso, dados são excluídos em até 30 dias, salvo obrigação legal. Consulte a seção 9 para detalhes.
