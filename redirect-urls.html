Configuração OAuth 2.0

URL de Autorização

https://www.linkedin.com/oauth/v2/authorization?response_type=code&client_id=77j88k6z6q9ta7&redirect_uri=YOUR_REDIRECT_URI&scope=openid%20profile%20email&state=YOUR_STATE_VALUE

URL de Token

https://www.linkedin.com/oauth/v2/accessToken

Escopos (Scopes) Aprovados

r_verificar openid profile w_member_social email r_profile_basicinfo

Descrição dos Escopos:

  • r_verificar: Obtenha a verificação do seu perfil
  • openid: Use seu nome e foto
  • profile: Use seu nome e foto
  • w_member_social: Crie, modifique e exclua publicações, comentários e reações em seu nome
  • email: Use o endereço de e-mail principal associado à sua conta do LinkedIn
  • r_profile_basicinfo: Acesse suas informações básicas de perfil (nome, título, foto, URL do perfil)

Duração do Token de Acesso

2 meses (5.184.000 segundos)

Exemplo de Fluxo OAuth 2.0:

  1. Usuário clica em "Login com LinkedIn"
  2. Redirecionado para URL de autorização do LinkedIn
  3. Usuário autoriza o aplicativo
  4. LinkedIn redireciona para seu redirect_uri com um código
  5. Backend troca o código por um access token
  6. Access token usado para acessar APIs do LinkedIn

Exemplo de Implementação (Python/Flask)

from flask import Flask, request, redirect
import requests
import os

app = Flask(__name__)

CLIENT_ID = "77j88k6z6q9ta7"
CLIENT_SECRET = os.getenv("LINKEDIN_CLIENT_SECRET")  # Variável de ambiente
REDIRECT_URI = "https://canalqb.com.br/callback"

AUTH_URL = "https://www.linkedin.com/oauth/v2/authorization"
TOKEN_URL = "https://www.linkedin.com/oauth/v2/accessToken"

@app.route("/")
def login():
    auth_link = (
        f"{AUTH_URL}"
        f"?response_type=code"
        f"&client_id={CLIENT_ID}"
        f"&redirect_uri={REDIRECT_URI}"
        f"&scope=openid profile email"
        f"&state=random_state_value"
    )
    return redirect(auth_link)

@app.route("/callback")
def callback():
    code = request.args.get("code")
    
    token_response = requests.post(
        TOKEN_URL,
        data={
            "grant_type": "authorization_code",
            "code": code,
            "redirect_uri": REDIRECT_URI,
            "client_id": CLIENT_ID,
            "client_secret": CLIENT_SECRET,
        },
    )
    
    token_data = token_response.json()
    access_token = token_data["access_token"]
    
    # Buscar perfil do usuário
    headers = {"Authorization": f"Bearer {access_token}"}
    profile = requests.get(
        "https://api.linkedin.com/v2/userinfo",
        headers=headers
    ).json()
    
    return profile

if __name__ == "__main__":
    app.run(debug=True)
